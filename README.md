# Vocabulary Explorer
This is a medium-high fidelity prototype for an app called Vocabulary Explorer, which aims to help users discover how words in the Plains Cree language relate to each other.

## Installation
To install and run this app, download the .apk file from the Releases tab of this page, and run it on an Android device.
